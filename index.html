Custom Low-Noise Arduino Design and Validation

Designed a custom Arduino-compatible PCB optimized to outperform commercial boards in terms of noise performance and signal integrity. Focused on clean schematic design, optimal layout with minimal cross-unders, and compact dimensions matching a commercial Arduino. Conducted comparative signal integrity tests—including rise/fall time and switching noise analysis—against a commercial board using slammer circuits and oscilloscope measurements. Identified layout-driven noise behaviors and proposed improvements, including repositioning decoupling capacitors for better power rail filtering.

Robotic Lane-Assist System with Custom Infrared Sensors

Developed and implemented a lane-assist system for an autonomous robotic platform. This project involved designing and fabricating custom infrared (IR) sensors for line detection, which interfaced with an Arduino microcontroller. The system's core functionality relied on processing input from these IR sensors to identify lane markings. A comparator circuit with adjustable sensitivity was incorporated to ensure reliable sensor performance across varied environmental conditions. The robot utilized this sensor data to execute programmed logic for course correction, gently guiding itself back into the lane. The project also featured visual and auditory alerts to indicate when the robot detected a lane boundary.
